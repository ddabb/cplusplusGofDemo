## 🚀 编译与测试

### 环境要求
- Visual Studio 2022（带 C++ 桌面开发工作负载）
- CMake 3.16+
- SFML 2.5（可选，用于 GUI 模式）

### 编译步骤

**方式1：使用 Visual Studio IDE**
1. 打开 VS 2022
2. 打开 CMake 项目（`CMakeLists.txt`）
3. 选择 `x64-Release` 配置
4. 生成 → 全部生成

**方式2：使用 Developer Command Prompt**

## 📝 技术笔记

### 圆角矩形实现
SFML 2.5 没有原生的圆角矩形类，使用以下方法组合实现：
- 中心矩形
- 左侧矩形
- 右侧矩形
- 四个圆角（圆形）

### 阴影效果实现
使用偏移的半透明矩形模拟阴影：
- 按钮阴影：底部偏移 3px
- 面板阴影：左上偏移 -3px，尺寸 +6px

### 性能考虑
- 圆形和矩形数量增加，但仍在可接受范围内
- 建议使用 Release 模式编译以获得最佳性能
